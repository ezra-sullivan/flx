# Tag Note: v0.1.7

**Date**: 2026-04-08

`v0.1.7` records the control-plane public package migration from the root facade into `github.com/ezra-sullivan/flx/pipeline/control`, plus the removal of the temporary root compatibility bridge.

## Changed

- Added the named public package `flx/pipeline/control` for worker options, error strategy, concurrency controller, and dynamic semaphore APIs.
- Updated root `flx` runtime, stream, transform, examples, and major docs to consume `pipeline/control` directly.
- Removed the temporary root compatibility bridge files `control_api.go` and `control_api_test.go`.
- Re-homed the remaining behavior tests into the runtime, stream, and transform test suites instead of keeping a compatibility-only test layer.

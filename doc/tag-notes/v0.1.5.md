# Tag Note: v0.1.5

**Date**: 2026-04-07

`v0.1.5` 当前作为 tag note 记录，而不是正式 release note。它对应一组 stage 语义封装落地、stage 风格示例重写，以及围绕同类型 stage 链式表达的 API 收口，用于冻结这一轮可读性与示例体验改进的集成点。

## Added

- New stage-family wrappers:
  `Stage`, `StageErr`, `FlatStage`, `FlatStageErr`, and `Tap`.
- Same-type chain helpers on `Stream[T]`:
  `Through`, `ThroughErr`, and method-form `Tap`.
- Regression coverage for the new stage-family and chain helpers in `stage_test.go`.

## Changed

- The HTTP image pipeline's stage-oriented example now uses `flx.Stage(...)` at cross-type boundaries and `Through(...)` across same-type stages.
- `README.md` and `doc/examples/http-image-pipeline.md` now document the stage-family APIs and the chained stage example style.
- The example documentation now matches the actual `.output/` path used by the runnable sample.

# Tag Note: v0.1.8

**Date**: 2026-04-08

`v0.1.8` records the first public pipeline observe/coordinator delivery, plus the worker-scaling policy hardening needed to support cooldown and hysteresis in real runtime decisions.

## Changed

- Added the named public packages `github.com/ezra-sullivan/flx/pipeline/observe` and `github.com/ezra-sullivan/flx/pipeline/coordinator`.
- Wired stage, link, and resource runtime state into real snapshots so observers can read live pipeline metrics instead of configuration-only data.
- Hardened coordinator policy behavior with separate scale-up and scale-down steps, cooldown windows, and hysteresis thresholds.
- Updated examples and core docs to show the coordinator-driven pipeline mode and the new observe/coordinator public surface.
